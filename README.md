# Share on IRC server

Pre-alpha stuff here!

# Installation

Install Redis.

Install libjpeg headers

    sudo apt-get install libjpeg8-dev

Run:

    npm install .

to install all deps.

Run the web server:

    coffee server.coffee

Run the irc bot:

    coffee ircbot.coffee


Client https://github.com/epeli/ircshare-android
